# 🌳 결정 트리 만들기 - 과학 교육용 웹앱

중학교 과학 수업을 위한 인터랙티브 결정 트리(Decision Tree) 제작 도구입니다.

## 📚 프로젝트 소개

학생들이 과학 개념을 체계적으로 정리하고 분류 알고리즘을 이해할 수 있도록 돕는 교육용 웹 애플리케이션입니다.

**활용 예시:**
- 태양계 천체 분류하기
- 생물 분류 체계 만들기
- 물질의 상태 변화 과정
- 화학 반응 종류 구분하기

## ✨ 주요 기능

### 1. 직관적인 블록 시스템
- **질문 블록**: 분류를 위한 질문 작성
- **답변 블록**: 최종 결과나 결론 작성

### 2. 자유로운 배치
- 드래그 앤 드롭으로 블록 이동
- 마우스와 터치 모두 지원

### 3. 데이터 관리
- 자동 저장 (localStorage)
- JSON 파일로 내보내기
- 페이지를 닫아도 내용 유지

### 4. 사용자 친화적 UI
- 깔끔하고 직관적인 디자인
- 모바일 반응형 지원
- 실시간 미리보기

## 🚀 시작하기

### GitHub Pages로 배포하기

1. **저장소 생성**
   - GitHub에서 새 저장소 생성
   - 저장소 이름: `decision-tree-maker` (원하는 이름으로 변경 가능)

2. **파일 업로드**
   ```
   decision-tree-maker/
   ├── index.html
   ├── styles.css
   ├── script.js
   └── README.md
   ```

3. **GitHub Pages 활성화**
   - 저장소 Settings → Pages
   - Source: Deploy from a branch
   - Branch: main (또는 master) → / (root) 선택
   - Save 클릭

4. **접속하기**
   - 5분 정도 후: `https://username.github.io/decision-tree-maker/`

## 📖 사용 방법

### 블록 추가하기
1. **질문 추가** 버튼 클릭 → 보라색 질문 블록 생성
2. **답변 추가** 버튼 클릭 → 초록색 답변 블록 생성

### 블록 편집하기
1. 블록 내부의 텍스트 영역 클릭
2. 질문이나 답변 입력
3. 입력 후 다른 곳 클릭 → 자동 저장

### 블록 이동하기
1. 블록의 헤더 부분을 클릭 또는 터치
2. 원하는 위치로 드래그
3. 놓으면 자동으로 위치 저장

### 블록 삭제하기
- 블록 우측 상단의 ✕ 버튼 클릭

### 전체 삭제하기
- **전체 삭제** 버튼 → 확인 → 모든 블록 제거

### 저장하기
- **저장** 버튼 → JSON 파일 다운로드

## 🎓 교육적 활용

### 1. 태양계 천체 분류
```
질문1: 스스로 빛을 낼까요?
  → 예 → 답변: 항성(태양)
  → 아니오 → 질문2: 행성 주위를 돌까요?
    → 예 → 답변: 위성(달)
    → 아니오 → 답변: 행성(지구)
```

### 2. 생물 분류
```
질문1: 척추가 있나요?
  → 예 → 질문2: 알을 낳나요?
    → 예 → 답변: 조류/파충류
    → 아니오 → 답변: 포유류
  → 아니오 → 답변: 무척추동물
```

### 3. 물질의 상태
```
질문1: 부피와 모양이 일정한가요?
  → 예 → 답변: 고체
  → 아니오 → 질문2: 부피는 일정한가요?
    → 예 → 답변: 액체
    → 아니오 → 답변: 기체
```
